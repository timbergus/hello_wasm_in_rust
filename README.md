# First WASM module with Rust

To build the WASM module, run the following command:

```bash
wasm-pack build --target web
```

## Prerequisites

[wasm-pack](https://github.com/rustwasm/wasm-pack)
[wasm-bindgen](https://github.com/rustwasm/wasm-bindgen)
